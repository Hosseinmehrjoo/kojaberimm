# 🧭 کجا بریم؟ — Kojaberam

> پلتفرم هوشمند گردشگری برای کشف مقصد و یافتن همسفر

---

## 📌 درباره پروژه

**کجا برم؟** یک پلتفرم گردشگری هوشمند است که به کاربران کمک می‌کند:

- مکان‌های گردشگری مناسب بر اساس بودجه، زمان و علایق خود پیدا کنند
- همسفر مناسب برای سفرشان بیابند
- برنامه سفر شخصی‌سازی‌شده دریافت کنند

---

## 🚀 قابلیت‌های نسخه اولیه (MVP)

- [x] نمایش مکان‌های گردشگری
- [x] فیلتر بر اساس بودجه، نوع سفر و مدت
- [x] سیستم معرفی همسفر
- [ ] برنامه‌ریزی خودکار سفر *(نسخه‌های بعدی)*
- [ ] تخمین هزینه *(نسخه‌های بعدی)*
- [ ] سیستم چت بین همسفرها *(نسخه‌های بعدی)*

---

## 🛠 تکنولوژی‌ها

| بخش | تکنولوژی |

## ⚙️ نصب و راه‌اندازی

### پیش‌نیازها

- Docker و Docker Compose نصب باشد
- Git نصب باشد

### اجرا

```bash
# کلون کردن پروژه
git clone https://github.com/your-username/kojaberam.git
cd kojaberam

# اجرا با Docker
docker-compose up --build
```

بعد از اجرا:
- Backend: `http://localhost:8000`
- Frontend: `http://localhost:3000`
